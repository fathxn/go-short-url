# go-shorten
Go Short URL is a simple backend RESTful API project built with some basic features like authentication, authorization, middleware, password encryption using Go Language Programming.

 *Features*
 - ✅ Basic auth with JWT
 - ✅ CRUD url shortener
 - ✅ Password encryption

*Tech stacks*
 - Fiber 
 - Gorm 
 - Bcrypt 
 - UUID
 - MySQL
 - Validator

*Tech stacks*
```console
go install
go run main.go
```
